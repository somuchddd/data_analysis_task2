# Аналитика данных. Задание 2.

## Описание задачи

Написать рабочий скрипт, который автоматически читает отзывы из внешнего файла, отправляет их в LLM и получает ответ в виде JSON файла, который содержит информацию о тональности и теме отзывов.

## Инструкция запуска

1. Склонировать репозиторий:
```
git clone https://github.com/somuchddd/data_analysis_task2.git
```
2. Установить зависимости:
```
pip install pandas openai python-dotenv
```
3. Зарегистрироваться на Groq и получить API-ключ

4. Создать файл .env в корне проекта с содержимым:
```
GROQ_API_KEY=*полученный_ключ*
```
5. Запустить скрипт

## Пример данных

Данные, которые использовались для тестирования скрипта находятся в файле data.csv, а результаты тестирования на этих данных - в result.json.

Пример входных данных (csv файл):
```
review
Отличный телефон! Камера супер, всё летает.
Качество ужасное, сломался через 3 дня.
Средненько, за такие деньги могло быть лучше.
```
